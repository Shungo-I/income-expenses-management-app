# 📊 収支管理アプリ

![Next.js](https://img.shields.io/badge/Next.js-15.3.3-black?style=flat-square&logo=next.js)
![TypeScript](https://img.shields.io/badge/TypeScript-5-blue?style=flat-square&logo=typescript)
![Tailwind CSS](https://img.shields.io/badge/Tailwind%20CSS-4-38bdf8?style=flat-square&logo=tailwind-css)

日々の収支を簡単に記録し、視覚的に把握できるモダンな家計簿・収支管理アプリケーションです。

## ✨ 実装済み機能

### 🏠 **ページトップ**

- **モダンなデザイン**: グラデーション背景とガラスモーフィズム効果
- **レスポンシブ対応**: デスクトップとモバイルの両方に最適化
- **ダークモード対応**: CSSクラスでダークモード準備完了（手動切り替え機能は未実装）

### 🧭 **ヘッダー**

- アプリケーションロゴとタイトル
- ナビゲーションメニュー（収入・支出・レポート）※UIのみ、機能は未実装
- スティッキーヘッダー（スクロール時も固定表示）

### 📈 **統計ダッシュボード（完全実装）**

- **今月の収入**: 緑色のカードで実際のデータを表示
- **今月の支出**: 赤色のカードで実際のデータを表示
- **残高**: 青色のカードで計算された残高を表示
- 各カードには対応するアイコン付き
- リアルタイムでの統計更新

### 📅 **インタラクティブカレンダー（完全実装）**

- 日本語表示（月名・曜日）
- 今日の日付のハイライト表示
- 日付選択機能
- 前月・次月のナビゲーション
- 選択された日付の詳細表示
- 取引がある日付のインジケータ表示
- スムーズなアニメーション効果

### 💰 **取引管理機能（完全実装）**

#### 🎯 **取引モーダル（TransactionModal）**
- **収入・支出タイプ選択**: 視覚的なタブ切り替え
- **金額入力**: 円マーク付きの数値入力フィールド
- **カテゴリー選択**: アイコン付きドロップダウンメニュー
  - 収入カテゴリー: 給与、ボーナス、フリーランス、投資、贈り物、その他収入
  - 支出カテゴリー: 食費、交通費、住居費、光熱費、娯楽、医療費、教育費、買い物、その他支出
- **説明入力**: 任意のメモ・詳細入力
- **日付選択**: カレンダーピッカー
- **バリデーション**: リアルタイムフォーム検証
- **編集モード**: 既存取引の編集機能
- **レスポンシブ対応**: モバイル・デスクトップ最適化
- **キーボード操作**: Escapeキーでドロップダウンを閉じる
- **アクセシビリティ**: スクリーンリーダー対応

#### 📋 **取引一覧（TransactionList）**
- **日付別表示**: 選択された日付の取引一覧
- **カテゴリーアイコン**: 各取引にアイコン表示
- **金額表示**: 収入（緑）・支出（赤）で色分け
- **編集・削除機能**: インライン操作ボタン
- **確認モーダル**: 削除時の確認ダイアログ
- **ソート機能**: 日付順でソート表示
- **レスポンシブデザイン**: モバイル対応レイアウト

#### 🗄️ **データ管理（useTransactions Hook）**
- **ローカルストレージ**: データの永続化
- **CRUD操作**: 作成・読取・更新・削除の完全実装
- **統計計算**: 月別・日別統計の自動計算
- **型安全**: TypeScriptによる完全な型定義
- **エラーハンドリング**: 適切なエラー処理

#### 🎨 **UI/UX改善**
- **確認モーダル**: 削除操作の安全な確認
- **ローディング状態**: データ読み込み中の表示
- **トランジション**: スムーズなアニメーション
- **アイコンシステム**: SVGアイコンによる視覚的表現

### 🚀 **クイックアクション（完全実装）**

- **収入を追加**: モーダル呼び出し機能付き緑色ボタン
- **支出を追加**: モーダル呼び出し機能付き赤色ボタン
- **レポートを見る**: 青色のグラデーションボタン（UIのみ）
- ホバー効果とシャドウアニメーション

### 🔧 **技術的実装**

#### **型定義システム**
- `Transaction`: 取引データ型
- `TransactionFormData`: フォームデータ型
- `TransactionType`: 収入・支出タイプ
- `TransactionCategory`: カテゴリー型
- 完全な型安全性とIntelliSense対応

#### **カテゴリーシステム**
- カテゴリーラベルの多言語対応（日本語）
- SVGアイコンセット
- 収入・支出別カテゴリー分類

## 🚀 開発・実行方法

### 前提条件

- Node.js 18以上
- npm または yarn

### インストール

```bash
# 依存関係のインストール
npm install

# 開発サーバーの起動
npm run dev

# ブラウザで http://localhost:3000 を開く
```

### ビルド・本番環境

```bash
# 本番用ビルド
npm run build

# 本番サーバーの起動
npm start
```

### コード品質・フォーマット

```bash
# コードフォーマットの実行
npm run format

# フォーマットチェック
npm run format:check

# ESLint実行
npm run lint
```
